</p>

_Just set ft=mtodo `set ft=mtodo`_

### 3 kind + heading

- `#` Heading
- `-` normal todo
- `x` done todo
- `*` important todo

_Todo items can be nested_

### 3 mappings

- `gd` mark done
- `gu` mark undone
- `gs` mark important (star)
